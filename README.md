# conversor-json-xlsx

Script que cria um arquivo .xlsx e insere dados json na tabela.
>Manipulação em um formato de Objeto específico.

### ex:

```
"employee": [
        {
            "id": "009475",
            "status": "enviado",
            "name": "qualquer nome",
            "dependents": [
                {
                    "id": 0.08882006578612167,
                    "name": "nome dependente",
                    "kinship": "primo",
                    "present": false
                }
             ]
        }
 ]
 ```
